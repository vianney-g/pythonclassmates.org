Installer Pelican

Published: Wed 09 January 2019
By Freezed

In Tutoriels.

tags: web python pelican

Démarrage rapide

C'est une traduction basique du démarrage rapide officiel

Créez le répertoire

mkdir -p ~/projects/yoursite
cd ~/projects/yoursite

Créez un environnement virtuel

virtualenv .venv -p /usr/bin/python3.6
source .venv/bin/activate

Installez Pelican et markdown (si besoin, optionnel)

pip install pelican markdown

Créez le squelette de l'arborescence

pelican-quickstart

Pour les questions avec une réponse entre crochet, appuyez sur Entrée pour acceptez cette valeur. Quand il vous sera demandé le préfixe de votre URL, entrez votre nom de domaine comme indiqué (http://example.com).

Créez un article

Vous ne pourrez lancer Pelican tant qu'il n'y a pas d'article. Utilisez votre éditeur préféré pour créez votre premier article avec le contenu suivant :

Title: My First Review
Date: 2010-12-03 10:20
Category: Review

Following is a review of my favorite mechanical keyboard.

Considerant que cet article est au format Markdown, enregistrez le en tant que ~/projects/yoursite/content/keyboard-review.md.

Générez le site

Toujours depuis le répertoire, lancez la commande Pelican pour générez votre site

pelican content

Votre site à été généré dans le répertoire output. Peut-être aurez vous un message d'alerte relatif aux flux, c'est normal en développement et vous pouvez l'ignorer

Prévisualisation du site

Ouvrez une session de terminal depuis le répertoire d'installation et lancez la commande suivante :

pelican --listen

Rendez vous à l'adresse http://localhost:8000/ à l'aide de votre navigateur, c'est parfait !
